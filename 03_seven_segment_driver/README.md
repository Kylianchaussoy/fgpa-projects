# Seven Segment Display Controller FPGA Project

## Overview
This project implements a seven-segment display controller using FPGA. The system receives a clock input and generates control signals for a four-digit seven-segment display. It sequentially cycles through a series of predefined characters, creating the effect of scrolling a message across the display. This is achieved by activating one segment at a time and changing the displayed characters based on a scrolling logic. The module includes clock dividers for managing display timing and segment activation.

## Board
This project is implemented on the Basys3 FPGA board.
