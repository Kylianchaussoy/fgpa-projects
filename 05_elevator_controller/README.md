# Elevator Controller FPGA Project
 
## Overview
This project implements a behavioral model of an elevator controller using VHDL. The elevator system simulates a 16-floor elevator, allowing users to call the elevator from any floor using slide switches. Each switch represents either a call to the elevator or an input for the final destination. Closing a switch indicates that a person has either entered or exited the elevator. The controller manages the state of the elevator, including movement, door operations, and LED indicators representing the current floor.
 
## Board
This project is implemented on the Basys3 FPGA board.
